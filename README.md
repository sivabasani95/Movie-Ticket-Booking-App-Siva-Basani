


# 🎬 BookMyScreen (BMS)

BookMyScreen is a React-based movie ticket booking UI that allows users to browse movies, view details, filter by language, and manage a wishlist.

## 🚀 Features

* 🎥 Browse recommended and all movies
* 🔍 Filter movies by language
* 📄 View detailed movie information
* ❤️ Add/remove movies from wishlist
* 📱 Responsive UI for all devices
* 🔄 Dynamic routing using React Router

## 🛠️ Tech Stack

* React.js
* React Router
* JavaScript (ES6+)
* CSS
* Vite
